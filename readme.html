Instalación

Instalación:

En esta implementación se ha utilizado una máquina ubuntu 24.04 y se ha instalado con la base de datos mysql-server

 

Black-forest

sudo apt update
sudo apt install bacula-server bacula-client mysql-server -y

Crear directorio para guardar backups:

sudo mkdir /backup
sudo chown bacula:bacula /backup # <button class="citation-flag" data-index="1">

Configuración del demonio de almacenamiento

sudo nano /etc/bacula/bacula-sd.conf

Editar los siguientes apartados:

Poner un nombre, el tipo en file y el directorio.

Verificar sintaxis:

sudo bacula-sd -tc /etc/bacula/bacula-sd.conf
